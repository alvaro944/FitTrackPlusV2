## Notas para decidir tiers

Pendiente de analisis cuando se acerque el lanzamiento. Criterios a considerar:

- **Limite de rutinas**: a partir de X rutinas → premium
- **Estadisticas avanzadas**: algunos graficos o el heatmap → premium
- **Alternativas de ejercicios**: posiblemente premium (feature diferencial)
- **Health Connect / pasos**: posiblemente premium
- **Historial ilimitado**: posiblemente el tier free limita a N sesiones
- **Funciones core** (registro de entrenamiento, 1-2 rutinas, historial basico) → siempre gratuitas

El analisis definitivo se hara con el usuario antes del lanzamiento.
