# Prompt Para El Siguiente Chat

```text
Quiero trabajar el branding de FitTrackPlus.

Contexto:
- App Android nativa de fitness en Kotlin + Jetpack Compose.
- Local-first.
- El producto ya tiene una direccion visual base en la UI, pero ahora quiero fijar la marca.
- No quiero empezar por dibujar un logo final a ciegas.
- Quiero definir primero personalidad, tono visual, paleta, tipografia y direccion de logo/app icon.

Lee primero estos archivos:
- docs/branding/README.md
- docs/branding/brand-foundation.md
- docs/branding/brand-questionnaire.md
- docs/branding/color-system.md
- docs/branding/typography.md
- docs/branding/logo-direction.md
- docs/branding/app-icon.md

Tu tarea:
1. Guiarme para cerrar la idea de marca.
2. Hacerme preguntas concretas, pocas y de alto valor.
3. Ayudarme a decidir si la marca debe ir por una linea mas:
   - atletica sobria
   - editorial premium
   - tecnica contenida
   - mezcla equilibrada
4. Proponer 2-3 direcciones de branding completas:
   - personalidad
   - paleta
   - tono
   - tipografia
   - direccion de logo
   - direccion de app icon
5. Recomendar cual merece explorarse primero.
6. Solo despues de cerrar eso, preparar un prompt para generar logo o icono con una herramienta de imagen.

Restricciones:
- Nada de branding fitness generico.
- Nada de culturismo agresivo.
- Nada de look SaaS aburrido.
- Nada de morado tipico ni neones.
- Quiero una marca util, sobria, moderna y con identidad.

Empieza revisando la base de marca y haciendome las preguntas minimas necesarias para decidir la direccion.
```
