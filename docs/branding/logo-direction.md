# Direccion De Logo

No es el logo final. Es la definicion de que tipo de logo merece explorarse.

## Estado

- Estado actual: cerrado
- Ultima decision valida: 2026-04-24 — simbolo abstracto seleccionado

## Regla

Primero idea. Luego forma. Despues imagen.

## Objetivo Del Logo

El logo debe poder convivir con:

- interfaz sobria
- icono de app
- posible portfolio o presentacion del proyecto

## Territorios Explorados

### 1. Wordmark sobrio

Descartado como ruta principal. Podria acompañar al simbolo en contextos de texto,
pero el simbolo funciona solo y es mas versatil para icono de app.

### 2. Monograma

Descartado. Las iniciales FT o FTP no aportan suficiente diferenciacion ni lectura semantica.

### 3. Simbolo abstracto

**Ruta elegida.**

## Decision Tomada

- **Ruta elegida**: simbolo abstracto.
- **Logo seleccionado**: `docs/branding/resources/Logo.png`.
- **Descripcion**: dos columnas/flechas ascendentes en negro grafito con un diamante verde
  en el apex. Contorno cobre/dorado en el diamante. Fondo crema neutro.
- **Lectura semantica**:
  - Las columnas = estructura y consistencia del entrenamiento
  - Las flechas hacia arriba = progreso y mejora continua
  - El diamante = calidad y valor que se acumula con el tiempo
- **Conexion con la app**: el verde del diamante conecta directamente con el color primario
  de la UI (#1F6B57). El cobre del contorno es el acento calido ya definido en el sistema de color.
- **Versatilidad confirmada**: funciona como simbolo solo, como icono de app, sobre fondos
  claros y oscuros, y puede escalar bien a tamaños pequeños.
- **No es wordmark**: el nombre puede acompañarlo en contextos de texto pero el simbolo
  funciona de forma autonoma.

## Cosas A Evitar

Nota adicional:

La intro de inicio puede acompanar el simbolo con wordmark y copy breve, pero ese uso no cambia la decision de marca principal: el logo sigue siendo el simbolo.

- mancuernas genericas
- rayos, llamas o signos de agresividad
- simbolos demasiado tech sin relacion con entreno
- marcas recargadas o demasiado complejas para icono
