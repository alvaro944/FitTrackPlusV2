# Direccion De Logo

No es el logo final. Es la definicion de que tipo de logo merece explorarse.

## Regla

Primero idea. Luego forma. Despues imagen.

## Objetivo Del Logo

El logo debe poder convivir con:

- interfaz sobria
- icono de app
- posible portfolio o presentacion del proyecto

## Territorios A Explorar

### 1. Wordmark sobrio

Encaja si queremos:

- marca madura
- look premium
- poca literalidad fitness

### 2. Monograma

Opciones:

- `FT`
- `FTP`

Encaja si queremos:

- buen icono
- sistema simple
- aplicacion facil a app icon

### 3. Simbolo abstracto

Ideas posibles:

- progreso
- bloque
- ritmo
- repeticion
- estructura

Encaja si queremos evitar:

- mancuernas
- barras
- pictogramas demasiado evidentes

## Cosas A Evitar

- mancuernas genericas
- rayos, llamas o signos de agresividad
- simbolos demasiado tech sin relacion con entreno
- marcas recargadas o demasiado complejas para icono

## Decision Pendiente

Elegir luego una de estas rutas:

- wordmark
- monograma
- simbolo abstracto
- sistema mixto
