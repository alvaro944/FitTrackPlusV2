# Branding FitTrackPlus

Esta carpeta recoge la construccion de la identidad visual y de marca de FitTrackPlus.

No sustituye al roadmap tecnico ni a `docs/work-methodology/`.

Su funcion es definir:

- personalidad de marca
- tono visual
- color
- tipografia
- direccion de logo
- icono de app
- criterios para futuras iteraciones visuales

## Objetivo

Dar coherencia a la app mas alla del redisenio de pantallas:

- que aspecto debe tener
- que sensacion debe transmitir
- que cosas debe evitar
- como se traduce eso a UI, logo e icono

## Orden Recomendado

1. `brand-foundation.md`
2. `brand-questionnaire.md`
3. `color-system.md`
4. `typography.md`
5. `logo-direction.md`
6. `app-icon.md`
7. `next-chat-prompt.md`

## Regla Practica

No empezar por un logo final.

Primero se fija:

- idea de marca
- atributos
- direccion cromatica
- tono
- lenguaje visual

Luego se explora:

- logo
- variantes
- icono
- posibles imagenes generadas
