# Preguntas De Marca

Responder esto antes de cerrar colores o logo.

## Identidad

1. Si FitTrackPlus fuera una marca real, que tres adjetivos deberian salir primero?
2. Que tres adjetivos no deberian salir nunca?
3. Debe sentirse mas como:
   - herramienta seria
   - companion personal
   - producto premium
   - sistema tecnico
4. Si tuvieras que compararla con un tipo de objeto fisico, seria mas:
   - libreta de entrenamiento premium
   - reloj deportivo limpio
   - panel tecnico
   - cuaderno editorial

## Tono Visual

5. Quieres una estetica mas:
   - calida
   - neutra
   - fria
6. Quieres que el color principal tenga mas peso o prefieres una base sobria con acentos puntuales?
7. Prefieres que el sistema visual sea:
   - mas minimal
   - mas marcado
   - mas expresivo

## Usuario

8. La app esta pensada visualmente mas para:
   - tu propio uso
   - portfolio
   - ambas por igual
9. Que sensacion debe tener alguien al abrirla por primera vez?
10. Que sensacion debe tener alguien al usarla en medio del gimnasio?

## Logo

11. Quieres un logo:
   - tipografico
   - simbolico
   - mixto
12. Debe apoyarse en:
   - iniciales
   - forma abstracta
   - barra/peso/movimiento
   - nada deportivo literal
13. Quieres que el logo sea mas:
   - sobrio
   - distintivo
   - editorial
   - tecnico

## App Icon

14. El icono debe parecer:
   - una app fitness
   - una app de productividad para entrenar
   - una app premium de tracking
15. Prefieres icono:
   - simbolo abstracto
   - inicial FT / FTP
   - forma geometrica inspirada en entrenamiento

## Referencias

16. Que marcas, apps o estilos visuales te atraen aqui?
17. Que marcas o estilos no quieres ni de lejos?
18. Si tuvieras que elegir una palabra-guia para el branding, cual seria?
